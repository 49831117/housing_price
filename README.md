# Housing Price

## 起源
- [內政部 實價登陸 Open Data](https://plvr.land.moi.gov.tw/DownloadOpenData)
- 觀察到某地區房價飆漲、薪水不漲的現象，思考是否各地皆如此。以此為出發點，期望可以透過此專案實踐將過去所學背景與程式語言結合，產出有用的數據分析資訊，提供未來多一個買房思考方向。
